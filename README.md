# Personal Expense Tracker using ReactJS

## Project Description
This project is a personal expense tracker built using ReactJS. It allows users to track their expenses, categorize them, and visualize their spending habits through charts and graphs. The primary goal is to provide individuals with a simple yet powerful tool to manage their finances effectively.

## Features
- **Add and Edit Expenses**: Users can add new expenses, and edit existing ones with ease.
- **Expense Categories**: Users can categorize expenses (e.g., Food, Transport, Entertainment) for better tracking.
- **Charts and Visualization**: The application provides visualizations of expenses using charts for insightful analysis.
- **Responsive Design**: Built with mobile-first principles, making it accessible on various devices.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/P-Jaswanth-Reddy/personal-expense-tracker-using-reactjs.git
   cd personal-expense-tracker-using-reactjs
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Usage Guide
Once the application is running, you can:
- Click on "Add Expense" to input a new expense.
- Use the "Edit" button next to each expense to modify it.
- View the visual representation of your expenses on the dashboard.
